BIOMARKER TO BV-BRC GENOME ID MAPPING PROJECT SUMMARY
=====================================================

Date: August 15, 2025
Project: Mapping biomarker names to BV-BRC genome IDs

OVERVIEW:
---------
This project involved extracting biomarker names from research data and mapping them to their corresponding BV-BRC genome identifiers for further analysis.

DATA SOURCES:
-------------
1. detailed_metrics_weighted_results.csv - Initial biomarker patterns file
2. romano_all_biomarkers.tsv - Horizontal list of 130 biomarkers  
3. romano_all_biomarkers_only.tsv - Comprehensive vertical list of 678 biomarkers
4. combo_top_biomarkers.tsv - Key biomarkers from research patterns

METHODOLOGY:
------------
1. Extracted biomarker names from CSV/TSV files
2. Converted horizontal biomarker lists to vertical format for processing
3. Queried BV-BRC API systematically to find genome IDs
4. Used multiple search strategies (Complete genomes, WGS, all statuses)
5. Researched alternative nomenclature for missing biomarkers
